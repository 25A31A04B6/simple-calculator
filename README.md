# simple-calculator

🧮 Simple Calculator in C

A basic calculator program written in C that performs arithmetic operations like addition, subtraction, multiplication, and division.

🚀 Features

- Supports +, -, *, / operations
- Handles division by zero
- Continuous calculations using loop
- User-friendly console interface

🛠️ Technologies Used

- C Programming Language
- Standard Input/Output Library

📌 How to Run

1. Compile the program:

gcc calculator.c -o calculator

2. Run the program:

./calculator

📷 Sample Output

=== Simple Calculator in C ===

Enter first number: 10
Enter operator (+, -, *, /): +
Enter second number: 5
10.00 + 5.00 = 15.00
